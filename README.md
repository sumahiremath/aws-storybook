# AWS Storybook

Short, story-driven notes for understanding AWS concepts.

This repo uses analogies and mental models to make cloud services easier to remember, compare, and explain.

Each note focuses on one concept at a time.

These notes are not official AWS documentation and are not a replacement for hands-on practice. They are simply learning aids for making abstract cloud ideas more concrete.

## Focus

* AWS fundamentals
* Serverless patterns
* Messaging and events
* Identity and access
* Databases
* Edge and delivery
* Developer Associate study topics

## Guiding idea

A good explanation should make the concept easier to retell.
